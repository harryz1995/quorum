Ollama not detected — quick setup

1. Install Ollama

curl -fsSL https://ollama.com/install.sh | sh # Windows: download from https://ollama.com/download

2. Pull models (mix different families for best debates)

ollama pull llama3.1 # Meta — great all-rounder ollama pull mistral # Mistral AI — sharp & concise ollama pull gemma2 # Google — analytical ollama pull phi3 # Microsoft — fast, small ollama pull deepseek-r1 # DeepSeek — strong reasoning

3. Start Ollama with CORS enabled (required for browser access)

# Mac/Linux OLLAMA_ORIGINS="*" ollama serve # Windows PowerShell $env:OLLAMA_ORIGINS="*"; ollama serve

Then click ↺ Refresh above.
